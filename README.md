# PROJECT TEMPLATE: Webpack + ReactJs + Firebase + Bootstrap

An empty project to start with.

## Getting Started

### Installing

Install NPM first

Then run:

```
npm install
```

To start the development server:

```
npm start
```

Build the bundled files (to be uploaded to your production server):

```
npm run-script build
```
